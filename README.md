
# Fixed Income Risk Analysis

This is a **course-based group project** completed as part of a graduate-level fixed income course. The project analyzes fixed income portfolio returns driven by **credit spread risk** and **interest rate risk**, and evaluates **delta hedging effectiveness** under different rebalancing frequencies.

## Key Features
- Credit spread & interest rate risk analysis
- SABR model calibration
- Swaption pricing using Black-76
- Delta hedging and PnL comparison
